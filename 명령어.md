# 명령어
바쁘신 분들은 [목차](http://pyrasis.com/Docker/Docker-HOWTO#section)에서 'search', 'pull', 'images', 'run', 'ps', 'start', 'attach', 'rm', 'rmi'만 보시고  
시간 있으시면 '사용하기'부터 끝까지 보시고 [이것도](https://www.leafcats.com/240) 보시는걸 추천드립니다.

--------------------------
명령어들의 자세한 옵션은  
`docker COMMAND --help`

다른 명령어들의 목록을 보려면
`docker`
